## 변경 유형 설명

- **추가됨**: 새로운 기능
- **변경됨**: 기존 기능의 변경사항
- **수정됨**: 버그 수정
- **제거됨**: 제거된 기능
- **보안**: 보안 관련 수정사항
- **성능**: 성능 개선사항

## 버전 형식

- **MAJOR**: 호환되지 않는 API 변경
- **MINOR**: 이전 버전과 호환되는 새로운 기능
- **PATCH**: 이전 버전과 호환되는 버그 수정
